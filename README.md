MelonJS is a dynamic, easy-to-use JavaScript runtime focused in experiments and data handling. Currently in development.

## Installation and usage:

- Via NPM:

```bash
npm install melon-runtime -g
```

```bash
npx melon [...commands]
```

- Inside the cloned repository:

```cpp
//Windows

.\com run [...commands]
```
